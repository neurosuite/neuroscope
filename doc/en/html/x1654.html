5.10. The Help Menu

Help->klusters Handbook

Opens this user manual in the Help Center

Help->What's this?

Activates the context help tool

Help->Report Bug

Helps you compose an e-mail message to report a bug to the author of Klusters

Help->About klusters

Shows information about the authors of and contributors to Klusters

Help->About KDE

Shows information about the authors of and contributors to KDE
